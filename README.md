# CyC Mobile Suite Releases

Repositorio publico para publicar las compilaciones Android de **CyC Mobile Suite**.

Este repositorio se usara exclusivamente como canal de distribucion de builds moviles, separado de la distribucion Desktop de CyC Topografia Suite.

## Uso previsto

- Crear una release por cada compilacion Android distribuible.
- Adjuntar el APK o AAB como asset de la release correspondiente.
- Registrar version, fecha, SHA-256 y notas importantes de instalacion.
- Mantener separada la linea Mobile de la linea Desktop.

## Convencion recomendada

Tags sugeridos:

```text
android-v0.1.0
android-v0.1.1
android-v0.2.0
```

Assets sugeridos:

```text
CyC_Mobile_Suite_v0.1.0.apk
CyC_Mobile_Suite_v0.1.0.aab
```
